# 2D-Game-Controller
## About
This is a case study project of the 2D Game Controller of the game, Celeste.

## Packages
* URP
* Cinemachine
* TextMeshPro
* Input System
* DoTween Free
* Naughty Attributes

## Controls
| Controls |    PC     | Playstation |    Xbox    |   Switch   |
|:--------:|:---------:|:-----------:|:----------:|:----------:|
|   Move   |   WASD    | Left Stick  | Left Stick | Left Stick |
|   Jump   |   Space   |  Cross (X)  |     A      |     B      |
|   Dash   | Shift/LMB | Square (▢)  |     X      |     Y      |
|  Attack  |     F     | Circle (◯)  |     B      |     A      |

## Effects
* Ghost Trail
* Particles
* Screen Ripple
